# Antigravity Skill Vault

Antigravity will read and use skills from these directories to accomplish complex tasks more effectively.

## Structure

- `.skills/`
  - `development/`
    - `fix_lint.md`: Strategies for lint and spell-check resolution.
    - `run_tests.md`: Instructions for running and debugging tests.
    - `deploy.md`: Deployment workflows.
  - `research/`
    - `meta_api.md`: Knowledge about the Meta WhatsApp API.
  - `README.md`: Overview of the Skill Vault.

## How to Use
You can mention a skill using `@skill-name` or simply tell Antigravity to use its expertise for a task.
